# Basic 2D Godot Project Template

This is a scaffold template for basic 2D Godot projects.

## Structure

- `project.godot` - Godot project configuration
- `scenes/` - Scene files
- `scripts/` - GDScript files
- `assets/` - Game assets (sprites, sounds, etc.)

## Usage

This template is used by the game-builder to scaffold new 2D game projects.
